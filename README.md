# express-calculator-api

## Install

    npm install

## Usage

### server

    npm start

### client

Send

    query={"op":"+", "ops":[2, 3]}

Get

    5

## Docker

Get the docker container

    docker pull juanpabloaj/express-calculator-api

Use

    docker run -p 8888:8888 juanpabloaj/express-calculator-api
